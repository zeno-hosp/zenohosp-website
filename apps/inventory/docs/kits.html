Kits (Bundles)

Bundle multiple items together for streamlined surgical dispensing and billing.

What is an Inventory Kit?

Often, medical procedures require a standard set of items. Instead of indenting or billing 15 different items individually every time a procedure happens, you can define a "Kit".

Example: C-Section Kit

You can create a Kit that contains:

  • 2 x Surgical Gloves
  • 1 x Scalpel
  • 5 x Gauze Swabs
  • 1 x Antiseptic Solution

Using Kits

When a nurse requests a "C-Section Kit" from the Main Store, the system will automatically deduct all the individual component items from the physical inventory in their respective quantities.

Similarly, when added to a patient's bill in the HMS, the patient is charged a single bundled price for the Kit, but the underlying inventory logs remain perfectly accurate.
