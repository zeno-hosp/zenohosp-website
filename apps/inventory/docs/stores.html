Stores (Locations)

Manage the physical and logical locations where inventory is held across the hospital.

Main Store vs Sub-Stores

Large hospitals operate a hub-and-spoke model for inventory to ensure departments don't run out of critical supplies during the night.

  • Main Store (Hub): This is usually where all external vendor deliveries (GRNs) arrive. It holds bulk stock.
  • Sub-Stores (Spokes): These are localized storage closets on specific floors or departments (e.g., ICU Store, 2nd Floor Pharmacy, OT Store).

Inter-Store Transfers (Indents)

When the ICU Store runs low on syringes, the Head Nurse raises an Indent Request to the Main Store.

The Main Store manager reviews the request, picks the stock, and processes a Stock Transfer. The system automatically deducts the items from the Main Store ledger and adds them to the ICU Store ledger, maintaining perfect traceability without affecting the global hospital stock count.

Warning: You cannot delete a Store if it currently holds active stock. You must transfer the stock out or write it off to zero before archiving the location.
