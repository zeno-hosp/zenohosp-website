Roles & Access

Understand how different user profiles access the ZenoHosp HMS system.

User Roles

The system comes pre-configured with three primary roles to ensure data security and operational efficiency:

  • Administrator (Admin): Has full access to the system. Can manage hospital settings, add new staff, configure billing rules, and view high-level revenue and patient dashboards.
  • Doctor: Focused access. Can view their own appointment schedules, access patient clinical records, write prescriptions, and log encounters. They cannot access hospital financial data or billing.
  • Receptionist: Front-desk access. Can register new patients, schedule appointments for all doctors, process OPD billing, and manage IPD admissions. They have limited access to clinical records (only demographics).

Logging In

Users access the system through the centralized login portal. Depending on the credentials entered, the dashboard will dynamically adjust the left-hand navigation menu to show only permitted modules.
