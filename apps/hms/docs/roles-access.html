How to Configure User Roles in HMS

Module: HMS Core • Level: Administrator

Role-Based Access Control (RBAC) is critical for hospital security. ZenoHosp allows you to create custom roles to ensure staff only see the patient data and modules they need.

Creating a New Role

Step 1: Navigate to Settings > User Management > Roles.
Step 2: Click the "+ Create New Role" button in the top right corner.
Step 3: Name the role (e.g., "Senior Duty Nurse") and assign it to a department.

Assigning Permissions

ZenoHosp uses a granular permission matrix. For your new role, you can toggle access for over 200 specific actions.

  • View EMR: Allows the user to read patient clinical notes.
  • Edit Vitals: Allows the user to input temperature, BP, and SpO2.
  • Billing Override: (Requires Admin PIN) Allows modifying an invoice post-generation.

Applying Roles to Users

Once a role is saved, go to Settings > User Management > Staff Directory. Select an employee profile and update their "Assigned Role" from the dropdown. Changes take effect on their next login.
