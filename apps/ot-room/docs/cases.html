Cases: Creating a New OT Booking

The end-to-end workflow for scheduling a surgical procedure, assigning surgeons, and preparing equipment.

1. Patient & Procedure Linking

To create a New OT Booking, start by searching for the Patient. Since ZenoHosp is integrated, you can search by Name, MRN, or Phone. The system will pull their details from the HMS. Next, select the specific surgical Procedure (e.g., Appendectomy, CABG) from your hospital's predefined Master list. The procedure selection often dictates default expected durations and required inventory.

2. Scheduling & OT Room Assignment

Select the proposed Start time and End time. Once the timeframe is set, the system will filter and display only the OT Rooms that are vacant and sterilized during that window.

3. Assigning the Surgical Team

Filter by specialization to assign the primary Surgeon. You can optionally add Anesthesiologists, Assistant Surgeons, and Scrub Nurses to the case. The system checks their availability against the main calendar to prevent double-booking.

4. Requesting Inventory Kits

Before confirming the booking, the surgeon or coordinator can request specific Inventory Kits (e.g., "Standard Ortho Implant Kit", "Cardiac Bypass Consumables"). This sends a direct, automated indent to the Inventory/Central Sterile Services Department (CSSD) so the equipment is prepped and carted to the theatre well before the start time.
