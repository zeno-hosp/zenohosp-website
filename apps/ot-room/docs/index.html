OT Room Dashboard

The central hub for monitoring Operation Theatre utilization, surgical efficiency, and overall theatre throughput.

Key Metrics & Overview

The dashboard provides OT Managers with a high-level summary of surgical operations:

  • Today's Surgeries: A count of completed, ongoing, and upcoming surgical cases scheduled for the current day across all available theatres.
  • Room Utilization: A percentage metric showing how effectively your OT rooms are being used versus their available block time, helping management decide if more theatres need to be opened or if existing blocks should be condensed.
  • Average Turnaround Time (TAT): Tracks the crucial minutes between one patient leaving the OT and the next patient entering, enabling you to optimize cleaning and prep processes.

Quick Actions

From the Dashboard, you can quickly jump into the Live Board to see real-time statuses or click New Booking to immediately start scheduling a new surgical case.
